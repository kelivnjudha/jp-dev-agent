RAW HID · EXPERIMENTAL HID raw capture reads the scanner directly in the main process, so scans work while JPPOS is focused. It is opt-in (JDA_HID_SCANNER_ENABLED) and auto-disables after a crash — Keyboard Wedge is the safe default. Keyboard-wedge scanners act like a keyboard and need this window focused.

    Only likely scanners are shown by default — keyboards, mice, Bluetooth receivers, and LED/RGB controllers are hidden behind "Show all". Device names, vendor/product ids, and masked serials only — raw HID reports and device paths never leave the agent core.

    If your scanner is shown as keyboard-mode, use the scanner vendor's setup barcode to switch it to HID-POS / USB HID POS mode for focus-independent scanning.
